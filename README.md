# GA3-Hersick-Saia-Marshall

### Description
Java MVC application using JDBC to perform full CRUD operations on a selected database table. Includes DAO-level unit tests (60%+ coverage) and SQL schema script.

### Requirements
- Maven
- Java JDK 21
- MySQL

### Setup Instructions
- Create a .env file following the format in [env example](.env.example)
- Run [SQL Initialization Script](Schema.sql) in MySQL Workbench
